# Skill: Note Underwriting

## Trigger
Use this skill when Kerry needs to analyze a mortgage note deal — from tape screen through IC memo. Built on the Desi Arnez methodology and the Standard Black Fund I buy box.

## Inputs Needed
- Deal details: UPB, property address (or state), lien position, payment status, asking price
- Tape or collateral file (paste data or describe what's available)
- Sleeve: Income (performing/reperforming) or Workout (non-performing)

---

## Buy Box Reference

### Income Sleeve (Performing / Reperforming)
- 1st lien only
- SFR and 2–4 unit residential; no condos preferred
- UPB: $40K–$350K (lower only if fixed costs still pencil)
- Min property value: ~$70K; as-is LTV target: 75% or less
- Performing: current, 9 of last 12 payments made
- Reperforming: 6+ consecutive on-time payments
- Escrowed taxes and insurance preferred

### Workout Sleeve (Non-Performing)
- 1st lien only; SFR and 2–4 unit only
- UPB: $30K–$300K; avoid low-value files where fixed costs absorb recovery
- Target bid: 60% or less of as-is value
- Every asset must have a defined exit path: cure/mod, settlement, deed-in-lieu, foreclosure already in progress, or clean REO exit
- Avoid: broken title, heavy litigation, broad bankruptcy complexity

---

## 8-Step Underwriting Workflow

### Step 1: Tape Screen
- Does it fit the buy box? (lien position, UPB range, property type, geography)
- Payment history — how many months current? Any gaps?
- Flag anything that disqualifies immediately
- Output: Pass / Conditional / Kill + reason

### Step 2: Collateral File & Enforceability Review
- Original note and mortgage/deed of trust present?
- Properly executed and recorded?
- Allonge chain of title intact (all endorsements in order)?
- Lost note affidavit needed?
- Output: Clean / Issues identified (list)

### Step 3: Conservative Valuation
- Pull comparable sales (comps) — 3–5 within 1 mile, similar SF and condition
- Use as-is value, not ARV
- Calculate as-is LTV: (UPB / As-Is Value)
- Flag if LTV exceeds 75% (Income) or 60% bid price (Workout)
- Output: Estimated value, LTV at asking price, max bid to stay in box

### Step 4: Title, Tax, Lien, Municipal, HOA & Probate Review
- Order or simulate title search — any senior liens?
- Property tax current or delinquent? How much owed?
- HOA dues — any super-priority lien risk by state?
- Code violations or municipal liens?
- Probate complications?
- Output: Clear / Issues list with estimated cure cost

### Step 5: Borrower & Servicing Reality Check
- Who is the borrower? Owner-occupied or investor?
- Bankruptcy status (active Chapter 7 or 13)?
- Death, divorce, or estate situation?
- Current servicer — licensed and cooperative?
- Output: Borrower profile summary + risk flags

### Step 6: Legal Timeline & Cost Model
- What state? (Judicial vs. non-judicial foreclosure)
- Estimated foreclosure timeline if needed
- Legal cost estimate (attorney fees, court costs)
- Eviction timeline if REO exit required
- Output: State-specific timeline, estimated legal cost budget

### Step 7: Financial Model
Run the numbers — build the model:

| Input | Value |
|---|---|
| UPB | |
| Asking price / bid | |
| Monthly P&I payment | |
| Interest rate on note | |
| Remaining term | |
| Property value (as-is) | |
| Outstanding taxes/liens | |

**Calculate:**
- Yield (if paying and holding to term)
- IRR (with projected exit or payoff timeline)
- Downside case: what's recovery if borrower defaults and foreclosure required?
- Max bid: what's the highest price that still clears the return threshold?

**Return targets (Fund I guidelines):**
- Income Sleeve: 10–14% yield target
- Workout Sleeve: 15%+ IRR target

Output: Model with base case, upside, and downside scenarios + max bid
