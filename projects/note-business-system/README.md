# Note Business Operating System

**Description:** Build a structured, AI-enabled operating system around the mortgage note business — SOPs, workflows, underwriting checklists, and automation where applicable.

**Status:** Planning — builds in parallel with mentorship

**Key Dates:**
- Initial SOP draft: Q2 2026
- Full system: Tied to Fund I launch readiness (Phase II)

## Scope
- Core SOPs: deal sourcing, tape screening, underwriting, IC memo, asset management
- AI integration points: first-pass underwriting support, IC memo drafts, research packaging, reporting templates
- Workflow documentation for all repeatable processes
- System designed so team members and AI can operate from the same frameworks
