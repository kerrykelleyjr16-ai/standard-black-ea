# Decision Log — The Pinnacle Note Fund

**Maintained By:** Agent 01 (Chief Operating Coordinator)
**Format:** Append-only. No entries are modified or deleted after creation.
**Purpose:** Institutional memory. Every material decision is logged here with date, context, reasoning, and approver.

---

## Log Format

```
[YYYY-MM-DD] | DECISION: [What was decided] | AGENT: [Which agent or who proposed] | DATA: [Key supporting data or reference] | APPROVER: [Human approver, if applicable] | STATUS: [Approved/Rejected/Pending] | NOTES: [Any additional context]
```

## Categories

Prefix log entries with the appropriate category code for easier filtering:

| Code | Category |
|---|---|
| ACQ | Acquisition — tape receipt, screening, bid, LOI, close |
| UW | Underwriting — classification, health score, conditional |
| PRICE | Pricing — bid recommendation, model assumption change |
| DUE | Diligence — exception, waiver, closing readiness |
| PORT | Portfolio — watchlist, workout transfer, monitoring |
| NPL | NPL — resolution strategy, legal action, REO |
| CASH | Cash — distribution, wire, reserve, capital call |
| RISK | Risk — limit breach, concentration, stress test |
| COMP | Compliance — material reviewed, cleared, policy change |
| GOV | Governance — conflict, related party, audit finding |
| VEND | Vendor — contract, termination, performance decision |
| IR | Investor relations — communication, subscription, redemption |
| OPS | Operations — general operating decision |
