## Purpose

The buy box defines the specific asset criteria that a loan must meet to be considered for acquisition. Agent 02 (Acquisitions) applies this screen to every tape received. Any deviation requires CEO/CIO approval.

---

## 1. Performing Note Criteria (Income Sleeve)

| Criterion | Requirement |
|---|---|
| Payment Status | Current or ≤30 days delinquent |
| Payment History | 12+ months continuous current history preferred; 9/12 months minimum |
| Coupon Rate | Market rate — below-market rates require pricing adjustment |
| Lien Position | 1st lien only — no 2nd liens in Fund I |
| UPB Range | $40,000 – $350,000 |
| LTV (at purchase) | ≤75% based on current appraised or BPO value |
| ITV (Investment-to-Value) | ≤65% (purchase price / current property value) |
| Property Type | SFR and 2–4 unit residential preferred; condos case-by-case (HOA review required) |
| Escrow | Escrowed preferred; non-escrowed permitted with tax and insurance confirmation |
| Tax Status | Current or delinquency ≤1 year with servicer cure plan |
| Insurance | Hazard insurance current; flood insurance current (if required) |
| Legal Status | No active foreclosure (prior resolved actions acceptable with 6 months clean) |
| Bankruptcy | No active bankruptcy (prior discharged acceptable with clean payment history) |

## 2. Reperforming Note Criteria (Income Sleeve)

| Criterion | Requirement |
|---|---|
| Payment Status | Current under modified or reinstated terms |
| Post-Modification Payment History | Minimum 6 months consecutive current payments after modification or reinstatement |
| Modification Terms | Must be reviewed and underwritten — rate, term, principal deferral, and forbearance amount documented |
| LTV (at purchase, current value) | ≤75% |
| ITV | ≤65% |
| Lien Position | 1st lien only — no 2nd liens in Fund I |
| Legal Status | No active foreclosure or bankruptcy |
| Escrow | Must be fully escrowed or confirmed direct-pay with documentation |
| Tax Status | Current |
| Insurance | Current |

---

## 3. Non-Performing Note Criteria (Workout Sleeve)

| Criterion | Requirement |
|---|---|
| Payment Status | Any delinquency level — 30 days through foreclosure |
| Lien Position | 1st lien only — no 2nd liens in Fund I |
| UPB Range | $30,000 – $300,000 |
| ITV (Investment-to-Value) | ≤60% standard; up to 70% for near-performing — this is the primary loss protection metric |
| LTV | Track; not a hard limit for NPLs — recovery via collateral is key |
| State | Judicial state exposure managed at portfolio level (see risk limits) |
| Legal Status | Any — foreclosure stage, bankruptcy, pre-foreclosure all eligible |
| Contactability | NPLs with zero contact history acceptable; factor into pricing |
| Environmental Risk | None known; exclude loans with confirmed environmental issues |
| Property Condition | All conditions acceptable; factor into pricing and recovery model |
| Senior Liens (2nd position) | Total senior lien + purchase price must still clear ITV hurdle |

## 10. Exclusion Rules

The following automatically exclude an asset from the buy box:

- Known environmental contamination on collateral
- Active title litigation with unsettled claims
- Third lien or more junior position
- Commercial real estate (non-residential primary use)
- Land only (no improvements)
- Manufactured home on leased land (title uncertainty)
- Loans in states where fund's servicer is not authorized
- Loans with known HOA super-priority lien that eliminates fund's interest
- UPB below fund minimum
- ITV above fund ceiling
- Any loan with active fraud indicators (confirmed)

---

## Exceptions

Any loan that does not meet buy box criteria requires CEO/CIO written approval before underwriting proceeds. Agent 02 must flag all exceptions at the tape screening stage. Exceptions are logged in `/logs/decision_log.md`.
