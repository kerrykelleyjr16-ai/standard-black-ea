# Table 5: Documents
## Requires: Deals + Contacts + Companies must be complete first

---

## Purpose
A registry of every document in the ecosystem — tape files, IC memos, LOIs, title reports, legal docs, SOPs, and templates. This is not file storage (Google Drive is). This is the index that tells you what exists, where it lives, and what deal it belongs to.

---

## How to Set Up

Click **+** → Add a table → Name it: **Documents**

---

## Fields — Build in This Exact Order

### Field 1: Document Name
- **Type:** Single line text
- **Primary field** — rename default "Name" to "Document Name"
- Be descriptive: "Evans Portfolio — Tape File 45 Assets" not just "Tape"

### Field 2: Document Type
- **Type:** Single select
- **Options:**
  - Tape File
  - IC Memo
  - LOI
  - Title Report
  - Legal Document
  - Servicer Agreement
  - BPO / Appraisal
  - Underwriting Model
  - Investor Report
  - SOP
  - Template
  - Course Material
  - Compliance
  - Other

### Field 3: Related Deal
- **Type:** Linked to another record
- **Link to table:** Deals
- Allow multiple: **No**

### Field 4: Related Contact
- **Type:** Linked to another record
- **Link to table:** Contacts
- Allow multiple: **No**

### Field 5: Related Company
- **Type:** Linked to another record
- **Link to table:** Companies
- Allow multiple: **No**

### Field 6: Storage Location
- **Type:** Single select
- **Options:**
  - Google Drive
  - Airtable Attachment
  - Notion
  - Local Only
  - Email

### Field 7: Drive Link
- **Type:** URL
- Paste the direct Google Drive link to the file

### Field 8: Attachment
- **Type:** Attachment
- For files small enough to attach directly in Airtable (PDFs, images under 5MB)

### Field 9: Status
- **Type:** Single select
- **Options:**
  - Draft
  - Final
  - Expired
  - Archived
  - Needs Review

### Field 10: Confidential
- **Type:** Checkbox
- Check for any document containing PII, investor data, or sensitive financial data

### Field 11: Last Updated
- **Type:** Date
- Format: M/D/YYYY

### Field 12: Notes
- **Type:** Long text

### Field 13: Date Added
- **Type:** Created time

---

## Views — Create These 5 Views

**View 1: All Documents** (rename the default)
- No filter
- Sort: Date Added DESC

**View 2: By Deal**
- No filter
- Group by: Related Deal
- Sort within groups: Document Type

**View 3: Active (Final)**
- Filter: Status = Final
- Sort: Last Updated DESC

**View 4: Needs Review**
- Filter: Status = Needs Review

**View 5: By Type**
- No filter
- Group by: Document Type

## Common Mistakes to Avoid

- **This is a registry, not storage** — never upload large tape files directly into Airtable. Use Drive Link to point to Google Drive
- **Confidential checkbox matters** — if this table ever gets shared with Kody or TJ, confidential documents should not be visible to them. Use it consistently
- **Status = "Final" means it's the authoritative version** — only one version of each document should be Final at a time
- **Related Deal is optional** — SOPs and templates don't belong to a specific deal; leave that field blank
